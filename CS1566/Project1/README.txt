HOW TO INSTALL/RUN:

1.) Place these files in their correct directories:

	A.) glut.h: ‘C:\Program Files (x86)\Microsoft Visual Studio 10.0\VC\include\GL\’
	B.) glut32.lib: ‘C:\Program Files (x86)\Microsoft Visual Studio 10.0\VC\lib\’
	C.) glut32.dll: ‘C:\Windows\SysWOW64\’
	
	OR, for 32-bit versions of Windows, you will want to do this:
	C.) glut32.dll: ‘C:\Windows\System32\’

2.) Once the above files have been properly placed, simply click on the .exe 

3.) Enjoy
